# Banking-Operations-Risk-Analytics-Dashboard
Tools Used: Power BI, DAX, Power Query (M), Data Modeling
## Project Overview
Developed a comprehensive 4-page business intelligence solution for a retail bank to monitor operational health, loan risk, and fraudulent activity. The project involved transforming raw, disconnected banking data into an interactive decision-making tool.

## Key Technical Challenges Solved
Complex Data Modeling: Rebuilt a fractured star schema by establishing a relationship between Branches → Customers → Accounts → Loans to ensure accurate filtering across all dimensions.

Data Integrity & Cleaning: Handled "Many-to-Many" relationship conflicts by performing duplicate removal and data type standardization in Power Query.

Advanced DAX Logic: Developed custom measures for risk assessment, including:

Default Rate: Tracking overdue principal vs. total lending.

Fraud Detection: Isolating transactions based on risk scores and high-value thresholds.

## Dashboard Breakdown
Banking Overview: High-level KPIs for total customers, transaction volume, and loan exposure.

Transaction Analysis: Deep dive into money movement trends and amount distribution.

Loan Risk Analysis: Monitoring the health of the loan portfolio by status and branch.

Fraud Detection: Security-focused page identifying "Large Transactions" and "Top Suspicious Accounts."
